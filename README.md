# App Gerenciamento (Kanban)

Aplicacao Kanban moderna em React + TypeScript, pronta para rodar via Docker.

## Rodar local sem Docker

```bash
npm install
npm run dev
```

## Rodar com Docker

```bash
docker compose up --build -d
```

Acesse: http://localhost:8085

## Parar container

```bash
docker compose down
```
